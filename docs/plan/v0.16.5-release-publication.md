# v0.16.5 Release Publication and Public Verification

Status: Completed.

## Goal

Publish the v0.16 release line from `main` after the completed v0.16.0-v0.16.4
mount/runtime hardening work, and verify the public artifacts and README
Quickstart against the published release.

## Scope

1. Confirm the release commit is merged to `main` before tagging.
2. Align Rust crate versions, TypeScript SDK package version,
   `PROTOCOL_VERSION`, CLI `--version`, README install examples, and release
   docs to the v0.16 release line.
3. Run main CI and CodeQL on the release commit.
4. Run the manual cross-platform live mount release gates for macOS FUSE-T and
   Windows WinFsp.
5. Build draft release artifacts through the release workflow.
6. Publish the release and verify public artifacts plus README Quickstart
   through GitHub Actions.
7. Record run IDs, tag, commit, and remaining limits in the phase plan.
