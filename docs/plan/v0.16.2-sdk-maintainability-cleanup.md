# v0.16.2 SDK Maintainability Cleanup

Status: Completed.

## Goal

Reduce the TypeScript SDK monolith risk while keeping the public SDK API and
generated gRPC surface unchanged.

## Scope

1. Extract SDK transport helpers from `packages/sdk-js/src/index.ts`.
2. Extract SDK gRPC request stream builders from `packages/sdk-js/src/index.ts`.
3. Preserve `OperonClient`, exported types, and generated runtime exports.
4. Validate TypeScript typecheck and SDK unit coverage.

## Completed

- Added `packages/sdk-js/src/transport.ts` for endpoint conversion, metadata
  construction, required-field checks, and BodyInit/stream byte helpers.
- Added `packages/sdk-js/src/grpc-requests.ts` for fs write, exec stdin,
  exec session, TCP service tunnel, and UDP datagram tunnel request streams.
- Kept `packages/sdk-js/src/index.ts` as the public API entrypoint and moved
  only behavior-preserving helpers out of the entrypoint.
- Added `scripts/verify-v0.16.2-sdk-maintainability-cleanup.sh` and wired it
  into the consolidated validation runner.

## Remaining

- No v0.16.2 SDK maintainability cleanup work remains.
